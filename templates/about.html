About This Project

The Spotify Song Popularity Prediction System is a machine learning-based web application designed to determine the potential popularity of songs based on their audio features. These features include variables such as:

  • Danceability
  • Energy
  • Loudness
  • Speechiness
  • Acousticness
  • Instrumentalness
  • Valence
  • Tempo
  • Duration
  • Chorus Hit
  • Sections
  • And more...

These inputs are processed and scaled using a trained data scaler, and then passed into a Machine Learning model (Logistic Regression / XGBoost) which predicts whether the song is likely to be popular or not.

How It Works

  1. User logs in or signs up
  2. Inputs song feature values into the predictor form
  3. System processes and predicts the result
  4. Displays whether the song is Popular or Not Popular

Key Features for Prediction

Feature Description
Danceability Measures how suitable a track is for dancing (0 to 1 scale).
Energy Represents the intensity and activity level of a track.
Loudness Measures the overall loudness in decibels.
Speechiness Determines the presence of spoken words in a track.
Acousticness Measures the degree of acoustic sounds in a song.
Instrumentalness Predicts whether a track is instrumental.
Valence Reflects the positivity or mood of a song.
Tempo Defines the speed of a song in beats per minute (BPM).
Time Signature Represents the musical time signature (e.g., 4/4, 3/4).
